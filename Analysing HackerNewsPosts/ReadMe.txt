This data collection contains Hacker News articles from the previous 12 months (up to September 26 2016).

The following columns are included:
title: the post's title (self explanatory)
num points: the number of upvotes the post received 
num comments: the number of comments the post received 
url: the url of the item being connected to
num points: the number of upvotes the post received 
num comments: the number of comments the post received
author: the name of the user who created the post.
Created at: the post's creation date and time (the time zone is Eastern Time in the US)

A model to predict the number of votes a post will receive.
